# Smart Inventory Management System

A simple inventory management system built using **Java, Spring Boot, MySQL, and Thymeleaf**.

## Features

* Add new products
* View all products
* Update product information
* Delete products
* Product search
* Dashboard with product statistics

## Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* Thymeleaf
* MySQL
* Bootstrap

1. Clone the repository

3. Run the Spring Boot application
4. Open browser at:

http://localhost:8080
